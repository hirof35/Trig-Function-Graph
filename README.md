# Trig Function Graph Simulator (三角関数グラフ・シミュレーター)
<img width="422" height="460" alt="スクリーンショット 2026-06-08 082358" src="https://github.com/user-attachments/assets/5c049e76-d229-4096-a57c-4814f836e830" />

Node.js（バックエンド）とHTML5 Canvas（フロントエンド）を組み合わせた、直感的でリアルタイムに動作する三角関数（sin/cos）の視覚化シミュレーターです。

スライダーを操作することで、単位円上における角度（$\theta$）、サイン（縦軸）、コサイン（横軸）の関係性を視覚的に学ぶことができます。

## 🚀 特徴

- **リアルタイム描画**: スライダーのドラッグと連動し、1ミリ秒の遅延もなくグラフが滑らかに更新されます。
- **バグフリーな描画**: Canvasのパスを適切に分離（`beginPath`）し、図形同士が意図しない線で繋がる描画バグを解消。
- **依存関係ゼロ**: 外部ライブラリ（Express等）を一切使わず、Node.jsの標準モジュール（http, fs, path）のみで軽量に動作します。
- **モダンなJS記述**: `var` を排除し、スコープの安全な `const` / `let` やイベントリスナー（`addEventListener`）を採用。

## 🛠️ 動作環境

- Node.js (v14以上推奨)
- モダンなWebブラウザ（Chrome, Edge, Safari, Firefoxなど）
